## Usage

### Documentation

Install pnpm:
[https://pnpm.io/installation](https://pnpm.io/installation)

Install dependencies:

```
pnpm install
```

Run development server:

```
pnpm docs:dev
```

Deploy

```
./deploy.sh
```
